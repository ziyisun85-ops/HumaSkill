```text
scripts/convert_g1_pkl_to_qpos.py
```

The script must:

```text
Read motions/metadata.json.
Use the same source_file mapping from Task 07.
Open each source .pkl file.
Extract root_trans_offset, root_rot, and dof.
Validate shapes:
  root_trans_offset must be [T, 3]
  root_rot must be [T, 4]
  dof must be [T, 23]
Build qpos:
  qpos = concat(root_trans_offset, root_rot, dof)

  qpos must be [T, 30]
Save qpos clips under motions/.
Create motions/metadata_qpos.json.
```
