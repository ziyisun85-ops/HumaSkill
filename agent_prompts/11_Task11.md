## 6. Playback Logic

### 6.1 Headless Playback

In headless mode:

```python
data.qpos[:] = motion[t]
mujoco.mj_forward(model, data)
```

Run through the clip once.

Do not open viewer.

### 6.2 Viewer Playback

In viewer mode:

```python
with mujoco.viewer.launch_passive(model, data) as viewer:
    ...
```

For each frame:

```python
data.qpos[:] = motion[t]
mujoco.mj_forward(model, data)
viewer.sync()
```

Use `viewer_fps` to control playback speed.

If `viewer_loop=True`, loop until the viewer is closed.

If `viewer_loop=False`, play once and exit.

Handle `KeyboardInterrupt` gracefully.
